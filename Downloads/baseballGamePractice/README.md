## 📖 프로젝트 README: 숫자 야구 게임 (MVC 패턴)

우테코 코딩 테스트 준비를 위한 **숫자 야구 게임** 콘솔 프로젝트의 요구사항 명세서입니다. 본 프로젝트는 **MVC (Model-View-Controller) 패턴**을 적용하여 객체 지향적인 설계 및 구현 능력을 연습하는 것을 목표로 합니다.

### ⚾️ 1. 게임 소개 및 규칙

컴퓨터가 임의로 생성한 3자리 숫자를 맞히는 콘솔 기반의 숫자 야구 게임입니다.

* **숫자 생성:** 1부터 9까지 서로 다른 숫자로 이루어진 **3자리** 임의의 숫자를 컴퓨터가 생성합니다.
* **게임 목표:** 3 Strike를 달성하여 정답을 맞히는 것입니다.
* **판별 기준:**
    * **스트라이크 (Strike):** 숫자와 위치가 모두 일치.
    * **볼 (Ball):** 숫자는 일치하지만 위치는 불일치.
    * **낫싱 (Nothing):** 일치하는 숫자가 전혀 없음.

| 예시 (정답: 421) | 사용자 입력 | 결과 |
| :---: | :---: | :--- |
| **Strike** | 421 | 3 스트라이크 (게임 종료) |
| **Ball** | 142 | 3 볼 |
| **Mix** | 452 | 1 스트라이크 1 볼 (4:S, 2:B) |
| **Nothing** | 567 | 낫싱 |

### ⚙️ 3. 기능 요구사항 상세

#### A. 게임 진행 및 종료

1.  게임을 시작할 때 **`숫자 야구 게임을 시작합니다.`** 메시지를 출력합니다.
2.  사용자에게 **`숫자를 입력해주세요 :`** 메시지와 함께 3자리 숫자를 입력받습니다.
3.  3 Strike를 달성하면 **`3개의 숫자를 모두 맞히셨습니다! 게임 종료`** 메시지를 출력합니다.
4.  게임 종료 후, **`게임을 새로 시작하려면 1, 종료하려면 2를 입력하세요.`** 메시지를 출력합니다.
5.  사용자가 `1`을 입력하면 새 게임을 시작하고, `2`를 입력하면 프로그램을 종료합니다.

#### B. 입출력 예시

(가정: 컴퓨터 정답은 `421`)

```
숫자 야구 게임을 시작합니다.
숫자를 입력해주세요 : 123
1 스트라이크 2 볼
숫자를 입력해주세요 : 421
3 스트라이크
3개의 숫자를 모두 맞히셨습니다! 게임 종료
게임을 새로 시작하려면 1, 종료하려면 2를 입력하세요.
1
숫자 야구 게임을 시작합니다.
... (새 게임 진행)
```

### 🚨 4. 예외 처리 (Exception Handling)

사용자 입력 값에 대한 유효성 검증을 Controller에서 수행하고, **유효하지 않은 입력이 들어올 경우 오류 메시지를 출력하고 프로그램을 즉시 종료**해야 합니다.

* **입력 값:**
    * 3자리 숫자가 아닌 경우 (예: '12', '1234').
    * 1\~9 범위를 벗어난 숫자가 포함된 경우 (예: '012', '109').
    * 중복된 숫자가 포함된 경우 (예: '112', '454').
    * 숫자가 아닌 문자 또는 공백이 포함된 경우 (예: 'a12', '1 2').
* **재시작/종료 선택:**
    * '1' 또는 '2' 외의 값이 입력된 경우.
